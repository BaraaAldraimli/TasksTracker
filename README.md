# TasksTracker
the project idea is to make a user-friendly task tracker web-application that makes it easy for people to manage their day, it has a main page containing the tasks with the ability to delete or edit them, by pressing the edit button it will direct you to the edit task page to edit it from there then press edit task, there is also a + icon that navigates to the add task page to keep adding more tasks.

#Problem:
people often forget their tasks, causing a problem managing their day activities that will eventually lead to somethings like missing deadlines or less productivity.


#Solution:
An interactive web-application that stores and views the tasks in a dashboard. making it easier for people to track their tasks to manage and organize the activities effectievely
