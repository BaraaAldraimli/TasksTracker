# TasksTracker
the project idea is to make a user-friendly task tracker web-application that makes it easy for people to manage their day, it has a main page containing the tasks in it with checkbox, and delete button, then navigate to add task page to keep adding daily tasks


#Problem
people tend to forget their daily tasks causing a problem with their day management


#Solution
A web-application to track their daily tasks to keep their day organized and well structured
